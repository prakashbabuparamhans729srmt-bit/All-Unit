# **App Name**: Aisha Browser

## Core Features:

- URL Input: Allow users to input a URL to visit a website, similar to Chrome.
- Page Display: Display the content of the website within the app, mirroring Chrome's rendering.
- Basic Navigation: Provide back and forward navigation buttons, just like Chrome.
- History Tracking: Maintain a history of visited pages, consistent with Chrome's history management.

## Style Guidelines:

- Color scheme should precisely match Chrome's default theme, including background and text colors.
- Font: Use the exact same font and sizes as Chrome for all UI elements to maintain visual consistency.
- Employ identical icons for navigation, settings, and other controls as Chrome, ensuring user familiarity.
- Replicate Chrome's layout with the URL bar at the top, tab structure, and menu placement to provide an indistinguishable experience.
- Implement the same subtle animations for loading, tab transitions, and other UI interactions as Chrome to mirror its behavior.
